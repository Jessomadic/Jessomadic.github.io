Media Transcoding Plugin

I wrote this JavaScript Plugin from scratch for the transcoding system known as Tdarr. This script allows high customizability for your desired media format including but not limited to
• Encoder - (Supports H.264,HVEC,H.265,AV1,8-bit and 10bit for all as well as CPU and GPU encoding)
• Bitrate - While most plugins and transcoding systems use a Quality profile that can give you a huge range of quality and bitrate this
allows you to set the bitrate that you know will look good and have that constent.
• Resolution Selection - This will let you Downscale the Resolution of your media to save tons of space while maintaining visual quality.
• Media Container - This allows you to select between MP4 or MKV depending on the compatibility you desire.
• Audio Type - This will transcode the audio track of the media file to the format of your choosing.
You can find this project on my Github
